# SystemVerilog Chisel to Quartus Transpiler

## Описание проекта

Транспилятор для преобразования кода, сгенерированного на Chisel, в совместимый SystemVerilog для Quartus II 13.1.0. Проект реализует автоматическую генерацию необходимых файлов проекта Quartus с использованием только стандартных инструментов Chisel и FIRRTL, без применения нейросетей.

## Требования

### Функциональные требования

1. Генерация Quartus проекта из Chisel кода
2. Создание .qsf файлов с параметрами FPGA
3. Генерация .sdc файлов с ограничениями тактовой частоты
4. Создание скриптов quartus_sh для синтеза/имплементации
5. Поддержка генерации .mif файлов из Scala данных
6. Аннотации критических путей через FIRRTL
7. Временное моделирование

### Нефункциональные требования

1. Совместимость с Quartus II 13.1 build 162
2. Поддержка Chisel 3.x и FIRRTL 1.x
3. Время генерации до 30 секунд
4. Потребление памяти до 2 ГБ
5. Поддержка YAML конфигураций устройств

## Зависимости

### Обязательные зависимости
- Scala 2.12+
- Chisel 3.x
- FIRRTL 1.x
- Quartus II 13.1
- Yaml support library

### Дополнительные зависимости
- ScalaTest 3.x (для тестирования)
- Mockito 4.x (для моков)
